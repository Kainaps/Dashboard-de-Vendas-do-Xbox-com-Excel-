## Dashboard de Vendas do Xbox – Excel

Este projeto apresenta uma dashboard de vendas do Xbox, criada para transformar dados brutos em informações visuais claras e úteis. O foco é permitir uma análise eficiente do desempenho de vendas e facilitar a tomada de decisões baseadas em dados.

## Ferramentas Utilizadas

Microsoft Excel

Tabelas dinâmicas

Gráficos dinâmicos

Segmentações de dados

Fórmulas para análise

## O que o Dashboard Mostra

Vendas totais

Produtos mais vendidos

Desempenho por período

Comparações e tendências

Principais KPIs

## Resultado

Um dashboard claro, organizado e eficiente, ideal para análises rápidas de vendas e como demonstração prática de habilidades em Excel e visualização de dados.
